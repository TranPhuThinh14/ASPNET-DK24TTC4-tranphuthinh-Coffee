# BÁO CÁO TIẾN ĐỘ TUẦN 2

## Thời gian
Tuần 2



## Kế hoạch đề ra
- Thiết kế database cho web bán cà phê
- Làm chức năng đăng nhập



## Kết quả thực hiện

### 1. Database
- Đã tạo database trên SQL Server
- Thiết kế bảng Users để quản lý tài khoản đăng nhập
- Thêm dữ liệu mẫu phục vụ kiểm thử

### 2. Chức năng đăng nhập
- Tạo trang Login.aspx
- Kết nối SQL Server bằng ADO.NET
- Kiểm tra đăng nhập từ bảng Users
- Xử lý đăng nhập sai / đúng

### 3. Điều hướng
- Sau khi đăng nhập thành công:
  - Chuyển sang trang Users.aspx
- Lưu trạng thái đăng nhập bằng Session



## Đánh giá
- Hoàn thành đầy đủ các nội dung theo kế hoạch
- Hệ thống hoạt động ổn định
- Sẵn sàng mở rộng phân quyền và chức năng mua hàng cho các tuần tiếp theo

- Phân quyền Admin / User
- Hiển thị danh sách sản phẩm cà phê
