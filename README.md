# CoffeeShop ASP.NET MVC

## Thông tin sinh viên
- Họ tên: Trần Phú Thịnh
- Lớp: DK24TTC4
- Môn học: ASP.NET
- Giảng viên hướng dẫn: TS. Đoàn Phước Miền

## Mô tả đồ án
Xây dựng website bán cà phê sử dụng ASP.NET Core MVC và SQL Server.
Hệ thống cho phép:
- Quản lý sản phẩm cà phê
- Đăng nhập / đăng ký người dùng
- Đặt hàng và quản lý đơn hàng

## Công nghệ sử dụng
- ASP.NET Core MVC
- SQL Server
- Entity Framework Core
- HTML, CSS, Bootstrap

## Tiến độ thực hiện
- Tuần 1: Cài đặt môi trường, tạo database, tạo project ASP.NET MVC
- Tuần 2: Thiết kế database, chức năng đăng nhập
- Tuần 3: Quản lý sản phẩm
- Tuần 4: Đặt hàng
- Tuần 5: Hoàn thiện & báo cáo

## Hướng dẫn chạy dự án
1. Clone repository
2. Mở bằng Visual Studio 2022
3. Cấu hình connection string
4. Run project
